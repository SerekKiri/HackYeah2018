# Installation

## Mobile Aplication

### For mobile app you need:
    - flutter
### Running application:
```
- cd app
```
```
- flutter packages get
```

    
```
- flutter run --release
```


***Remember to turn on USB Debugging on your mobile phone***

## :building_construction: Backend
### For backend you need:
    - docker
    - docker compose
### Running backend:

```
- docker compose up
```
